Resumo de Comandos

Comando Descrição
git init Inicializa um novo repositório Git local.
git add . Adiciona todas as mudanças à área de staging.
git commit -m "msg" Salva o snapshot no histórico do projeto.
git remote add origin Vincula o repositório local a um servidor remoto.
git push -u origin Envia os commits locais para o repositório remoto.
