O Commit

Um commit é um "snapshot" permanente do seu projeto no histórico do Git.

Ele registra quem fez a alteração, quando e, o mais importante, o que foi alterado através da mensagem.

Dica: Escreva mensagens claras e no imperativo, como "Adiciona funcionalidade X" ou "Corrige bug Y".

# Salva as mudanças do staging:
git commit -m "Mensagem descritiva"
