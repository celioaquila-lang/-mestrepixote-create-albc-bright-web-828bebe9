Staging e .gitignore

Área de Staging

Prepare seus arquivos para o próximo commit. O Git permite selecionar exatamente o que será salvo.

# Adiciona tudo
git add .

# Adiciona arquivo específico
git add index.html
→ Funciona como um "rascunho" antes da versão final.

O Arquivo .gitignore

Evite enviar arquivos desnecessários ou sensíveis para o repositório.

# Exemplo de .gitignore
node_modules/
.env
*.log
.DS_Store
→ Mantém o repositório limpo e seguro.
